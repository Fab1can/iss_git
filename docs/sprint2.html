Problem analysis

Non è necessario implementare un nuovo algoritmo per il gioco, è sufficiente utilizzare la libreria prodotta nello sprint precedente ed implementare l'interfaccia IOutDev, che si occuperà di gestire l'interazione tra il dominio del gioco e l'interfaccia grafica Java Swing.
L'interfaccia è stata implementata dalla classe ConwayLifeGridClaude.
