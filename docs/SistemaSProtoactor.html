Deployment

Il server Javalin è stato containerizzato con Docker secondo R4. La sua immagine viene costruita con il comando "./gradlew distTar && docker build -t sistemasprotoactor-1.0 .", e viene eseguita con il comando "docker compose -f sistemasprotoactor.yaml up".
Dopo aver eseguito il container si potranno vedere dai log della console i protoattori interagire fra di loro.
